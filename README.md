## General info
A "More Places to Stay" image carousel

## Technologies
Project is created with:
* PostgreSQL
* Express
* React
* Node.js
* Webpack

## Setup
To run this project:
1) Start (or Install) PostgreSQL
2) Update configuration files
3) Install it locally using npm:

```
$ cd carousel-service
$ npm install
$ npm start
```
